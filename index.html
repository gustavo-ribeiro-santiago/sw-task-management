Rioops Task Manager

The Rioops Task Manager is a comprehensive task management system developed from 2019 to 2020 using Microsoft Access and VBA for seamless integration and efficient task tracking within a large shipping company. This tailored solution was designed to streamline, automate and organize various operational tasks, enhancing productivity and ensuring timely deliveries.

Welcome

The application features login authentication with role-based access controls, allowing different teams to access only the information relevant to their responsibilities.

Home Menu

From the home menu, the user can access his tasks, view the last changes in the company's Voyage Management System (Dataloy), change the application settings and exit.
